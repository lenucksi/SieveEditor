---
description: Package the application as a JAR
---

Build and package SieveEditor as an executable JAR with dependencies.

Execute: ` mvn clean package`

After successful packaging:
1. Confirm JAR location: `target/SieveEditor-jar-with-dependencies.jar`
2. Show file size
3. Provide run command: `java -jar target/SieveEditor-jar-with-dependencies.jar`
